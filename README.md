# Mezquita Al‑Quds del Vendrell  
Plataforma web para la gestión interna y la comunicación pública de la Mezquita Al‑Quds del Vendrell.

## Descripción general
Este proyecto proporciona una solución completa para la mezquita, permitiendo gestionar y mostrar información relevante para la comunidad:

- Horarios de rezo con contador automático  
- Gestión de donativos y facturas  
- Publicación de noticias  
- Información del imán  
- Información de clases de árabe y Corán  
- Notificaciones importantes  
- Formulario de contacto  
- Botón de ubicación (Google Maps / Apple Maps)

Incluye un panel administrativo exclusivo para el imam.

## Documento de requerimientos
El documento completo de requerimientos (funcionales, no funcionales, diseño visual y modelo de datos) está disponible en la carpeta **/docs** del repositorio.

## Tecnologías utilizadas
- **Laravel** – Backend  
- **React** – Frontend  
- **TailwindCSS** – Estilos  
- **MySQL** – Base de datos  
- **Vite** – Compilación de assets
